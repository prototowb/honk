## Conventions In Force
- **Build origin**: change a tool → `spmc-server/lib/tools.js`; change a limit → `lib/specs.js`; change metadata/version → `spmc-server/package.json`. Then `npm run build`. Never hand-edit generated artifacts (`TOOLS.md`, `.claude-plugin/plugin.json`, and — after B/A — `skills/*`, README/CONTEXT tables, MCP configs).
- Server `spmc`; `run.js` = MCP only; `start.js` = MCP + scheduler. One dispatcher (`lib/dispatch.js`); every real publish goes through `publishAudited`.
- Credentials: `~/.claude/spmc.env` (self-loaded by `run.js`); multi-account `KEY__ACCOUNT`.
- Keep the suite green at every commit. Runtime deps still 2; build tooling adds **zero** (root `package.json` has no dependencies).
