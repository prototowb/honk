## Conventions In Force
- **Build origin**: change a tool → `lib/tools.js`; a limit → `lib/specs.js`; a credential/media key → `lib/config.js` (or `specs.js` credentials); metadata/version → `spmc-server/package.json`. Then `npm run build`. Never hand-edit generated artifacts (`TOOLS.md`, `.claude-plugin/plugin.json`, the `gen:tools` regions in `README.md` + `hermes/CONTEXT.md`, the 3 MCP configs, and — after B2 — `skills/*`). Hand-editing **outside** the `gen:tools` markers is fine.
- Server `spmc`; `run.js` = MCP only; `start.js` = MCP + scheduler. One dispatcher (`lib/dispatch.js`); every real publish goes through `publishAudited`.
- Credentials: `~/.claude/spmc.env` (self-loaded by `run.js`); multi-account `KEY__ACCOUNT`.
- Keep the suite green at every commit. Runtime deps still 2; build tooling adds **zero**.
