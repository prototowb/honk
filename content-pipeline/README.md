# Content Pipeline Plugin v1.2

Complete social media content automation: **manual ideas OR trend research** → concepts → review → platform-specific content → queue → schedule → publish.

## Skills

1. **Submit content idea** — Manually provide an idea with metadata
2. **Research topic trends** *(NEW)* — Auto-research trending topics (trends, news, social)
3. **Run content pipeline** — Execute: concept generation → review → content creation
4. **Manage content queue & schedule** — Dashboard, schedule, publish, analytics

## Workflows

### Path A: Manual Idea (Traditional)

```
/Submit content idea
  → Describe your idea (topic, audience, tone, references)
  ↓
/Run content pipeline
  → Concepts generated → Reviewed → Content created
  ↓
/Manage content queue & schedule
  → Review, approve, schedule, publish
```

### Path B: Trend Research (Automated)

```
/Research topic trends
  → Analyzes trending topics (Google Trends, Reddit, news, social)
  → Selects promising angle
  → Generates idea brief
  ↓
/Run content pipeline
  → Concepts generated → Reviewed → Content created
  ↓
/Manage content queue & schedule
  → Review, approve, schedule, publish
```

## Scheduling (Both paths)

Use Cowork's scheduler to automate:

**Manual ideas on schedule:**
```
Every Monday 9am:
  /Submit content idea → [scheduled prompt with topic]
  /Run content pipeline → [auto-execute]
```

**Trend research on schedule:**
```
Every day 8am:
  /Research topic trends → [auto-research]
  /Run content pipeline → [auto-execute]
  
Result: Daily trending content queued & ready to review
```

**Mix both:**
- Trends daily (auto-generated queue)
- Manual ideas 2x weekly (when you have specific angles)
- All feed same pipeline & queue

## Output per idea

- **3 concepts** with title, angle, key message, hook
- **Review feedback** on originality, audience fit, brand alignment
- **Final content** ready for all 3 platforms:
  - Instagram: Long caption + 15-30 hashtags + alt text
  - X: Tweet thread + mentions + alt text
  - TikTok: Video hook + caption + trending hashtags
